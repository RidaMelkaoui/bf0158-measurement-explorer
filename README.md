# BF0158 Measurement Explorer

Interactive dimensional measurement explorer for Mounting Spring BF0158.

The website reads only the dataset derived from `CLIPSP1.xlsx`, sheet `modif 3`. It includes all 51 samples, all 16 characteristic test IDs, and all 816 measurement values.

Use **Export PDF** to download a three-page, meeting-ready report for the currently selected characteristic and active filters. The report includes the headline KPIs, tolerance chart, exact measurement table, conformance summary, and out-of-spec map.

## Open locally

Open `index.html` through a local static web server. For a no-install alternative, use the self-contained `BF0158_Interactive_Measurement_Explorer.html` deliverable.

## Data scope

- 51 samples
- 16 characteristic test IDs
- 816 measured values

The two source rows named `Angle 3` are kept separate as Test 11 and Test 12.
